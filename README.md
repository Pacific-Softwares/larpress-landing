# larpress.Digital Landing Page

A modern, responsive landing page for **larpress.Digital** - a Laravel CMS and application starter kit.

## Features

- Modern responsive design with dark mode toggle
- Hero section, features, pricing, and installation guide
- Privacy policy and terms & conditions pages
- Built with HTML, Tailwind CSS (CDN), and vanilla JavaScript

## Project Structure

```
larpress-landing/
├── index.html                 # Main landing page
├── privacy-policy.html        # Privacy policy
├── terms-and-conditions.html # Terms & conditions
└── src/
    ├── admin.png             # Admin panel screenshot
    └── larpress-logo.svg     # Logo
```

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required.

For deployment, upload all files to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).

## Customization

- Update content directly in HTML files
- Modify gradient colors in CSS (default: `#667eea` to `#764ba2`)
- Replace images in the `src/` directory
